### YouTube Video Predictions

Create prediction markets based on YouTube video performance metrics like view count, likes, or comments.
Settlement: Automatic via YouTube API integration

### Adding YouTube Videos

Simply paste any YouTube video link to create a prediction market about its future performance.

<Frame>
  <img src="/images/yt1.png" alt="Interface showing where to paste YouTube links" style={{ borderRadius: '0.5rem' }} />
</Frame>


### Setting Prediction Parameters

Choose what to predict about the video - views, likes, comments, or other engagement metrics.
Select a timeframe for your prediction - end of day, week, or month.


<Frame>
  <img src="/images/yt2.png" alt="Interface showing where to paste YouTube links" style={{ borderRadius: '0.5rem' }} />
</Frame>


### Market Settlement

Markets automatically settle using our YouTube API integration when the prediction timeframe ends.

<Frame>
  <img src="/images/yt3.png" alt="YouTube market settlement process" style={{ borderRadius: '0.5rem' }} />
</Frame>


### Popular Use Cases

- Predicting viral potential of new uploads
- Forecasting subscriber milestones
- Estimating engagement for upcoming premieres
- Predicting performance of trending videos
