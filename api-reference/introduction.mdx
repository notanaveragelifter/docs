## PNP Exchange API

Build on top of PNP Exchange with a simple, secure REST API for markets, outcomes, trades, and settlements. The full reference is generated from our OpenAPI spec.

### Base URL

```
https://api.pnp.exchange/v1
```

### Authentication

All requests require a Bearer token.

```
Authorization: Bearer <PNP_API_KEY>
```

Example:

```bash
curl -H "Authorization: Bearer $PNP_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.pnp.exchange/v1/markets
```

### Errors

Errors follow RFC 7807-style problem details where possible.

```json
{
  "type": "https://api.pnp.exchange/errors/unauthorized",
  "title": "Unauthorized",
  "status": 401,
  "detail": "Missing or invalid API key"
}
```
